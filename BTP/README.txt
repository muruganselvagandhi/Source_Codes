## Project Overview

This repository contains the complete development files for the Bachelor Thesis Project titled **“Development of an Electric Vehicle System with Integrated Battery Management and IoT-Enabled Monitoring.”**

The objective of this project is to design and implement an electric vehicle (EV) system integrated with a Battery Management System (BMS) and an IoT-enabled monitoring platform for real-time data acquisition, visualization, and performance analysis of battery and vehicle parameters.

---

## Repository Structure

The repository is organized into three main directories as described below:

### 1. Dash/

This folder contains all files related to the **IoT dashboard**.

* Responsible for real-time visualization of battery and vehicle parameters
* Acquires data from the cloud platform
* Displays parameters such as voltage, current, state of charge (SOC), and other BMS metrics
* Serves as the primary user interface for monitoring system performance

---

### 2. Design/

This folder contains the **mechanical and system design files**.

* Includes the electric vehicle design developed using **Autodesk Fusion 360**
* Represents the physical layout and structural design of the vehicle
* Used for design validation and visualization of the overall EV system

---

### 3. Uno_Code/

This folder contains the **embedded firmware** for the system.

* Arduino Uno code for interfacing with the Battery Management System
* Acquires battery and vehicle parameters from the BMS
* Displays real-time parameters on an OLED display
* Sends collected data to the cloud for IoT-based monitoring and visualization

---

## Project Highlights

* Integrated Battery Management System for electric vehicle applications
* Real-time battery and vehicle parameter acquisition
* OLED-based local monitoring interface
* Cloud-connected IoT monitoring system
